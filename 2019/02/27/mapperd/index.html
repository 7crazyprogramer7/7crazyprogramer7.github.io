mybatis的Mapped Statements collection does not contain value for 问题解决

1. 命名空间没写对，如果使用mapper，则

<mapper namespace="com.sitech.mapper.StudentMapper">

#####空间中一定要写上Mapper的名称student一定要写)，否则就会报错。

2. 原因是mybatis.cfg.xml配置文件缺少了对Mapper.xml的引用加上

<mapper resource="cn/cq/mapper/SchoolCardMapper.xml"/>

3. mapper.xml文件名和所写的接口名称不相同
4. 接口中没有此方法
5. 若使用

<package name="cn.cq.mapper"/>
一定在mapper包下加

此图中的shoolcardmapper.java文件

6. 或者直接用

<mapper resource="cn/cq/mapper/StudentMapper.xml"/> <mapper resource="cn/cq/mapper/SchoolCardMapper.xml"/>

使用Hexo

1. hexo的简单使用

  • hexo是什么？

    heox is A fast, simple & powerful blog framework ，我们通过hexo搭建的博客然后部署到 github 上 ，别人通过 github 主页可以访问到你搭建的博客。

  • 前期准备

    • 首先安装一个 git 和 注册一个 github 账号(可点击此处自行安装)
    • 然后安装 Node.js 。（请自行安装，过程很简单）

  • 安装 Hexo （注意：以下所要输入的皆为命令行）

    1. 在D盘中（建议在非系统盘）建一个文件夹
      • 在文件夹中点击右键，选择git bash here.
      • 然后输入 npm install -g hexo-cli 即可。
    2. 再输入hexo init 进行初始化，然后输入hexo install 安装其他一些可用的文件包。
    3. 安装 deployer-git 插件：继续输入npm install hexo-deployer-git –save

  • 建立GitHub项目

    为项目命名，格式为（用户名 + github.io）

  • 修改_config.yml 文件

    • 在GitHub上复制生成的地址
    • 打开_config.yml文件,改deploy下的type后为git，repo后为复制的地址，branch后为manster
  • 再输入hexo g -d, 然后在网页上输入https://你的github用户名.ginhub.io即可。
